# 🚀 Como Iniciar o Aliria RH

## Opção 1️⃣: Clique no Atalho do Desktop (MAIS FÁCIL)
1. Procure no Desktop por: **"Aliria RH.lnk.cmd"**
2. Clique 2x
3. Uma janela preta abrirá com o servidor rodando
4. Acesse: http://localhost:5173

## Opção 2️⃣: Arquivo Batch (Se o atalho não funcionar)
1. Navegue até: `C:\Klissia - RH\rh-app\`
2. Clique 2x em: **iniciar-app.bat**
3. Uma janela preta abrirá com o servidor rodando
4. Acesse: http://localhost:5173

## Opção 3️⃣: Command Prompt (Manual)
1. Abra Command Prompt (CMD)
2. Digite:
```
cd C:\Klissia - RH\rh-app
npm run dev
```
3. Acesse: http://localhost:5173

---

## ✅ Quando o App está Rodando
Você verá na janela preta:
```
VITE v8.0.14 ready in XXX ms
➜  Local:   http://localhost:5173/
```

## ❌ Se Não Funcionar
1. Feche a janela preta (Ctrl+C)
2. Tente novamente
3. Se persistir, chame Klissia

---

**Dica:** Deixe a janela preta aberta enquanto usar o app!
