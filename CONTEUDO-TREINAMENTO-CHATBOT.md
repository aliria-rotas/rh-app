## 📚 Módulo 1: Fundamentos do Atendimento em Chatbot (20 min)

### 1.1 Por que o Chatbot é Importante?

**Contexto**: O chatbot é muitas vezes o **primeiro contato** do paciente com a Aliria.

```
Paciente
   ↓
Chatbot (primeira impressão)
   ↓
Atendimento humano (se necessário)
```

### 1.2 As 3 Pilares do Bom Atendimento em Chatbot

| Pilar | Descrição | Exemplo |
|-------|-----------|---------|
| **Clareza** | Mensagens diretas, sem ambiguidade | ❌ "Pode ser" / ✅ "Sim, pode realizar o cadastro em 5 minutos" |
| **Empatia** | Reconhecimento de sentimentos | ❌ "Problema resolvido" / ✅ "Entendo sua preocupação. Vou ajudar!" |
| **Eficiência** | Respostas rápidas e precisas | ❌ 10 mensagens / ✅ 2-3 mensagens claras |

## 📚 Módulo 2: Como Falar e Escrever (30 min)

### 2.1 Tom de Voz Ideal

**Use um tom:**
- ✅ **Profissional** - Competente, seguro
- ✅ **Amigável** - Acessível, personalizado
- ✅ **Respeitoso** - Formal quando necessário
- ✅ **Empático** - Compreensivo das dificuldades

**Evite:**
- ❌ Tom robótico ou frio
- ❌ Gírias excessivas
- ❌ Jargão técnico sem explicação
- ❌ Instruções complexas

### 2.2 Estrutura de Mensagens

**Boa estrutura:**

```
[Saudação] + [Compreensão] + [Ação] + [Oferta adicional]

"Olá! Entendo que você precisa de ajuda com o cadastro. 
Vou orientá-lo passo a passo. Depois, se tiver mais dúvidas, 
é só chamar!"
```

### 2.3 Exemplos Práticos

#### ❌ ERRADO
```
"Cadastro de paciente: insira CPF, nome completo, data de 
nascimento, telefone, email, endereço. Máximo 100 caracteres."
```

#### ✅ CERTO
```
"Ótimo! Vamos começar seu cadastro. 

1️⃣ Qual é seu CPF? (11 dígitos)
2️⃣ Seu nome completo?

Pode responder com calma! 😊"
```

---

### 2.4 Linguagem Clara e Acessível

**Regra**: Escreva como se estivesse falando com um amigo.

| ❌ Evitar | ✅ Usar |
|-----------|---------|
| "É imperativo que você realize o upload do documento" | "Por favor, compartilhe o documento com a gente" |
| "Não é possível efetuar a transação neste momento" | "No momento não conseguimos processar. Tente novamente em 5 minutos" |
| "Requisitar suporte técnico" | "Precisa de ajuda? Me avisa!" |
| "Procedimento não autorizado" | "Essa ação não é permitida. Posso ajudar de outra forma?" |

## 📚 Módulo 3: Gentilezas e Empatia (20 min)

### 3.1 Palavras-Chave de Gentileza

**Use frequentemente:**

```
Por favor, obrigado, desculpa, entendo, claro, 
sem problema, com prazer, fico feliz em ajudar, 
deixa comigo, relaxa, estou aqui para você
```

### 3.2 Frases Mágicas

#### Para Começar
```
"Olá! Bem-vindo à Aliria! 👋 Como posso ajudá-lo hoje?"
"Oi! Tudo bem? O que você precisa?"
"Que legal ter você aqui! Em que posso ser útil?"
```

#### Para Entender
```
"Entendi sua dúvida. Deixa eu ajudar."
"Ótima pergunta! Vou esclarecer isso para você."
"Compreendo suas preocupações. Vamos resolver juntos."
```

#### Para Pedir Desculpas
```
"Desculpa pelo transtorno! Deixa comigo."
"Entendo que isso foi frustrante. Vou corrigir."
"Mil desculpas! Não deveria ter acontecido."
```

#### Para Encerrar
```
"Fico feliz em ter ajudado! 😊 Qualquer coisa, é só chamar!"
"Obrigado por conversar comigo! Vou ficar feliz se precisar novamente."
"Tudo resolvido? Ótimo! Volte sempre!"
```

### 3.3 Empatia em Ação

**Quando o paciente está frustrado:**

```
❌ ERRADO:
"A política é essa, deve seguir as regras."

✅ CERTO:
"Entendo sua frustração. Essa é uma situação chata, e eu 
teria a mesma preocupação. Deixa eu ver o que posso fazer 
para melhorar isso. Um momento!"
```

**Quando o paciente está com dúvida:**

```
❌ ERRADO:
"É simples. Basta clicar em 'cadastro'."

✅ CERTO:
"Sem problema! Essa é uma dúvida muito comum. Vou guiar 
você passo a passo. É bem fácil, pode deixar!"
```

## 📚 Módulo 4: Boas Práticas de Comunicação (30 min)

### 4.1 Tamanho das Mensagens

**Regra**: Uma ideia por mensagem. Máximo 3 linhas.

```
❌ ERRADO (texto grande demais):
"Olá, para realizar o cadastro você deve preencher os campos 
com seus dados pessoais incluindo CPF, RG, data de nascimento, 
telefone, email e endereço completo, e depois clicar em enviar 
para que o sistema processe o pedido"

✅ CERTO (quebrado em partes):
"Olá! Vamos fazer seu cadastro? 😊

Preciso de alguns dados:
- CPF
- Nome completo
- Telefone

Pode começar?"
```

### 4.2 Velocidade de Resposta

| Tempo | Mensagem |
|-------|----------|
| 0-2 seg | "Oi! 👋" (reconhecer presença) |
| 5-10 seg | Resposta com informação |
| Mais de 30 seg | "Um momento, deixa eu verificar..." (manter conectado) |

### 4.3 Personificação

**Use nomes e contexto:**

```
❌ "Seu pedido foi recebido"
✅ "Maria, seu pedido #12345 foi recebido! Vamos processar em breve."

❌ "Medicamento indisponível"
✅ "Infelizmente o Dipirona não está disponível agora, mas temos 
Novalgina como alternativa. Quer que eu te ajude com isso?"
```

### 4.4 Chamada para Ação (CTA) Clara

**Sempre deixe claro o próximo passo:**

```
"Pronto! Você pode agora:
1️⃣ Finalizar o cadastro
2️⃣ Consultar medicamentos
3️⃣ Falar com um atendente

Qual você prefere?"
```

## 📚 Módulo 5: Situações Delicadas (20 min)

### 5.1 Paciente com Dor ou Sofrimento

```
"Sinto muito que você esteja passando por isso. 😟 
Entendo como é difícil lidar com [o problema].

Vou ajudá-lo da melhor forma possível. 
O que você precisa agora?"
```

### 5.2 Paciente Frustrado ou Irritado

```
"Entendo sua frustração, e acho válida sua reclamação. 
Você tem razão em se preocupar.

Vou ver o que posso fazer para resolver. 
Pode deixar comigo!"
```

### 5.3 Paciente com Dúvida Repetida

```
"Sem problema! Essa é uma dúvida comum e importante.
Deixa eu esclarecer melhor dessa vez..."
```

### 5.4 Quando Você Não Sabe a Resposta

```
❌ "Não sei"
✅ "Ótima pergunta! Vou verificar com meu time para 
te dar a resposta correta. Você pode aguardar um momento?"
```

## 📚 Módulo 6: Checklist de Qualidade (10 min)

Antes de enviar sua mensagem, pergunte-se:

- [ ] **Clareza**: Alguém que não sabe do assunto entenderia?
- [ ] **Empatia**: Demonstra preocupação com o paciente?
- [ ] **Brevidade**: Tem menos de 3 linhas? (ou está bem estruturado)
- [ ] **Tom**: Profissional mas acessível?
- [ ] **Próximo passo**: Fica claro o que fazer depois?
- [ ] **Gentileza**: Tem palavras de gentileza?
- [ ] **Sem erros**: Sem typos, bem-formatado?
- [ ] **Pessoal**: Usa o nome do paciente ou referência pessoal?

## 🎓 Conclusão

### Pontos-Chave a Lembrar

1. **Clareza acima de tudo** - O paciente deve entender de primeira
2. **Empatia é seu superpower** - Conecte-se com o sentimento, não só a dúvida
3. **Gentileza custa nada** - Use palavras mágicas sempre
4. **Brevidade é ouro** - Uma boa mensagem é concisa
5. **Personalize** - Use nome, contexto, histórico
6. **Próximo passo claro** - Sempre deixe óbvio o que fazer
7. **Humanize a máquina** - Lembre que é um HUMANO atrás do chatbot
